# PROJECT_PLAN.md — CareerFit AI 프로젝트 기획 초안

## 1. 문제 정의
1. 어떤 스킬을 준비해야 할지 모른다
2. 비슷한 공고들을 비교하기가 너무 귀찮다
3. 내 이력서가 이 직무에 맞는지 모르겠다
4. 어떤 직무가 있고, 어떤 역량을 길러야 할지 모르겠다

## 2. 사용자 정의
- 대상: 취업을 희망하는 대학교 학부생

- 사용 시나리오:
1. 사용자가 전공, 보유 스킬, 관심 직무를 입력한다.
2. CareerFit AI가 데이터를 검색해 역량 매칭 결과를 반환한다.
3. 어떤 데이터를 기반으로 답변했는지 출처를 함께 표시한다.

## 3. 데이터 계획
- 취업 공고 컬럼: 회사명, 직무명, 요구 스킬, 우대 사항, 지원 자격, 업무 내용, 마감일
- 공모전 컬럼: 공모전명, 주최, 분야, 참가 자격, 요구 역량, 상금, 마감일
- 데이터 출처: 강사 제공 목업 CSV (3일차)

## 4. 기능 목록
| 우선순위 | 기능 | 구현 일차 | 필수/선택 |
|---|---|---|---|
| 1 | 역량 매칭 분석 | 4일차 | 필수 |
| 2 | 부족 역량 추천 | 4일차 | 필수 |
| 3 | RAG 기반 근거 답변 | 4일차 | 필수 |
| 4 | 출처 표시 | 4일차 | 필수 |
| 5 | 기타 사항 | 4일차 | 선택 |
